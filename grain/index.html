Bienvenue

Ce jeu est une version simplifé du célèbre Tetris dans lequel vous pouvez affronter un ordinateur jouant en utilisant un algorithme dit de contrôleur optimal

Pour jouer, cliquez là où vous voulez placer la pièce et utilisez les boutons pour la tourner. Plus de détails sur les règles sur la page de documentation .
